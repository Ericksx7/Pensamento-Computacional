# Explicação da Atividade

Nesta atividade, nosso grupo desenvolveu a modelagem do projeto **“Plataforma Inteligente de Disponibilidade de Medicamentos”**, utilizando conceitos de **Pensamento Computacional** e representações visuais de sistemas. O principal objetivo foi organizar e demonstrar o funcionamento da plataforma através de três perspectivas diferentes: **Lista, Hierarquia e Grafo**.

Cada modelo foi utilizado para representar os mesmos dados de formas diferentes, permitindo compreender melhor como os módulos do sistema se organizam e se conectam.

# Objetivos da Atividade

- Aplicar os conceitos de pensamento computacional;
- Representar um sistema através de diferentes modelos visuais;
- Compreender a estrutura e organização dos módulos;
- Demonstrar as relações entre os componentes do sistema;
- Auxiliar no planejamento do desenvolvimento da plataforma;
- Desenvolver habilidades de modelagem e organização de sistemas.
